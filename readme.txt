Easily add the Visual Website Optimizer script on your website.

== Description ==

This plugin let's you easily add the Visual Website Optimizer (VWO) script to all your Wordpress pages. Simply copy/paste the Account_id from the experiment on [vwo.com](https://vwo.com/) into the input field, save settings and you're up and running.

Yes, it's as easy as that.

== Installation ==

1. Upload the downloaded plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add your Visual Website Optimizer account_id to the settings page and hit "Save settings".
4. PROFIT!
